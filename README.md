# PTP-102 Visual Aid

Equine laminitis visual aid tool with X-ray upload, severity grading, and reference comparison.

## Setup

1. Copy `.env.example` to `.env` and fill in your Supabase credentials
2. Open `public/index.html` in a browser or serve with a static server

## Features

- Reference X-ray viewer with severity overlays (Normal/Mild/Moderate/Severe)
- Patient X-ray viewer with horse selection and date navigation
- Upload new X-rays to Supabase Storage
- Clickable pins with anatomical tooltips
- Responsive design for desktop and tablet

## Tech Stack

- HTML/CSS/JS (vanilla, no build step required)
- Supabase (Auth, Database, Storage)
- Deployed on [hosting platform]
